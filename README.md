# App-Download

iFunBox App 下载页面

## 简介

这是一个精美的 iOS 应用下载页面，使用 HTML5 + CSS3 构建，包含以下特性：

- 🎨 现代简约的深色主题设计
- ✨ 蓝色发光效果和呼吸动画
- 📱 响应式布局，适配移动端
- 🚀 平滑的页面载入动画
- 🔗 社交链接（GitHub、Telegram）

## 使用方法

1. 直接在浏览器中打开 `indeex.html` 文件
2. 或部署到任何静态网页托管服务

## 自定义

- 修改 `indeex.html` 中的应用名称和链接
- 替换 `logo.png` 为你的应用图标
- 调整 CSS 样式自定义颜色和效果
